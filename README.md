# at-coder-js

use JS for AtCoder

## Before using

npm i

pip3 install --user online-judge-tools
oj login https://beta.atcoder.jp/
npm install -g atcoder-cli
acc login
acc session

## setup

rewrite config.js

## compile

npm run build

## test

npm run test

## submit

oj s https://atcoder.jp/contests/abc149/tasks/abc149_a src/abc149_a/compile.js

実装途中
npm run submit
